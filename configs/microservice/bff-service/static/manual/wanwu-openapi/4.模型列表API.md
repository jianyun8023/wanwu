# 模型列表 API


该接口用于根据筛选条件查询当前 API Key 所属用户/组织可见的模型列表。未传 `isActive` 时默认仅返回已启用模型。
### 请求接口
| Type                 | Instructions                                                                 |
| -------------------- | ---------------------------------------------------------------------------- |
| 方法                 | Http                                                                         |
| 请求URL              | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/model/list` |
| 字符编码             | UTF-8                                                                        |
| 请求类型             | GET                                                                          |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}`                                            |
| 请求格式             | Query 参数（URL查询参数）                                                     |
| 响应格式             | `Content-Type: application/json`                                             |
### 请求参数
| Parameter   | Required | Type   | Instructions                                                                 |
| ----------- | -------- | ------ | ---------------------------------------------------------------------------- |
| modelType   | 否       | string | 模型类型：`llm`、`embedding`、`rerank`                                       |
| provider    | 否       | string | 模型供应商（例如：`YuanJing`）                                                 |
| displayName | 否       | string | 模型显示名称（支持按名称筛选）                                               |
### 响应参数
| Parameter | Type   | Instructions                                                 |
| --------- | ------ | ------------------------------------------------------------ |
| code      | int    | 状态码，用于表示请求成功或具体的错误类型                     |
| msg       | string | 提示信息，通常用于提供关于code的详细解释或请求成功的确认信息 |
| data      | object | 响应数据主体                                                 |
#### data
| Parameter | Type  | Instructions |
| --------- | ----- | ------------ |
| list      | array | 模型列表     |
| total     | int64 | 模型总数     |
#### list[]
| Parameter   | Type   | Instructions |
| ----------- | ------ | ------------ |
| uuid        | string | 模型UUID     |
| displayName | string | 模型显示名称 |
| provider    | string | 模型供应商   |
| modelType   | string | 模型类型     |
| model       | string | 模型名称     |
| scopeType   | string | 模型公开范围 |

### 调用示例
```shell
curl -k --location --get 'http://localhost:8081/service/api/openapi/v1/model/list' \
--data-urlencode 'provider=YuanJing' \
--data-urlencode 'modelType=rerank' \
--header 'Accept: application/json' \
--header 'Authorization: Bearer <API Key>'
```
