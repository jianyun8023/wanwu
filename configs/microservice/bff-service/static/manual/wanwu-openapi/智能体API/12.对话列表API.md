# 对话列表API

获取指定智能体的**已发布对话**列表，按创建时间降序分页返回。草稿态对话不在此接口范围内（草稿态每个智能体只有一条会话，参见 [16.草稿态对话历史API.md](16.草稿态对话历史API.md)）。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/conversation/list` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| uuid | 是 | string | 智能体UUID |
| pageNo | 是 | int | 页码，从 1 开始 |
| pageSize | 是 | int | 每页条数，从 1 开始 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| list | array | 对话列表 |
| total | int | 总数 |
| pageNo | int | 当前页码 |
| pageSize | int | 每页条数 |

### data.list[]

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| conversationId | string | 对话ID，用于对话历史/删除等接口传参 |
| assistantId | string | 智能体内部ID |
| title | string | 对话标题 |
| createdAt | string | 创建时间 |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/conversation/list?uuid=2004010127610351679&pageNo=1&pageSize=20' \
--header 'Authorization: Bearer <API Key>'
```

## 响应示例

```json
{
    "code": 0,
    "msg": "",
    "data": {
        "list": [
            {
                "conversationId": "56",
                "assistantId": "78",
                "title": "你好，DeepSeek",
                "createdAt": "2026-05-07 10:15:00"
            }
        ],
        "total": 1,
        "pageNo": 1,
        "pageSize": 20
    }
}
```
