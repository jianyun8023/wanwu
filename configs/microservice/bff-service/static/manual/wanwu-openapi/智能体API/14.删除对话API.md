# 删除对话API

删除指定**已发布对话**：移除 conversation 主体及其所有历史消息，conversation_id 不再可用。

> - 若仅希望**清空历史但保留 conversation_id** 继续使用，请改用 [15.清空对话历史API.md](15.清空对话历史API.md)。
> - 若希望**仅删除某一条消息**，同样使用 [15.清空对话历史API.md](15.清空对话历史API.md)（传入 `detail_id`）。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/conversation` |
| 字符编码 | UTF-8 |
| 请求类型 | DELETE |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | 参数走 query string，无请求体 |
| 响应格式 | `Content-Type: application/json` |

## 请求参数（query）

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| conversation_id | 是 | string | 对话ID |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 空对象 |

## 调用示例

```shell
curl -k --location --request DELETE 'http://localhost:8081/service/api/openapi/v1/agent/conversation?conversation_id=56' \
--header 'Authorization: Bearer <API Key>'
```

## 响应示例

```json
{
    "code": 0,
    "msg": "",
    "data": null
}
```
