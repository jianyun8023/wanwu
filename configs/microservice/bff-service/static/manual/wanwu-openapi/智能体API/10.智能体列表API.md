# 智能体列表API

获取当前 API Key 所属用户下的全部智能体列表（含草稿态与已发布态），支持按名称模糊筛选。响应中的 `uuid` 可用于其他智能体接口传参。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/list` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| name | 否 | string | 按名称模糊筛选 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| list | array | 智能体列表 |
| total | int | 总数 |

### data.list[]

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| uuid | string | 智能体UUID，后续接口传参 |
| name | string | 智能体名称 |
| desc | string | 智能体描述 |
| avatar | object | 头像信息 |
| category | int | 智能体类型：1-单智能体，2-多智能体 |
| publishType | string | 发布范围：`public`/`organization`/`private`，空字符串表示未发布（草稿态） |
| version | string | 已发布版本号，未发布时为空 |
| createdAt | string | 创建时间 |
| updatedAt | string | 最后更新时间 |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/list?name=助手' \
--header 'Authorization: Bearer <API Key>'
```
