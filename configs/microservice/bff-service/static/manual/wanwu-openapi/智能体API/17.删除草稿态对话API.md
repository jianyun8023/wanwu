# 删除草稿态对话API

删除指定智能体的**草稿态**对话历史。草稿态每个智能体仅维护一条会话，因此通过 `uuid` 定位。
- 传入 `detail_id`：仅删除该条消息
- 不传 `detail_id`：清空全部草稿历史（草稿会话本身保留，可继续调用[草稿态对话API](3.草稿态对话API.md)）

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/conversation/draft` |
| 字符编码 | UTF-8 |
| 请求类型 | DELETE |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | 参数走 query string，无请求体 |
| 响应格式 | `Content-Type: application/json` |

## 请求参数（query）

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| uuid | 是 | string | 智能体UUID |
| detail_id | 否 | string | 消息ID；不传时清空全部草稿历史 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 空对象 |

## 调用示例

### 清空全部草稿历史

```shell
curl -k --location --request DELETE 'http://localhost:8081/service/api/openapi/v1/agent/conversation/draft?uuid=2004010127610351679' \
--header 'Authorization: Bearer <API Key>'
```

### 删除单条草稿消息

```shell
curl -k --location --request DELETE 'http://localhost:8081/service/api/openapi/v1/agent/conversation/draft?uuid=2004010127610351679&detail_id=2001' \
--header 'Authorization: Bearer <API Key>'
```

## 响应示例

```json
{
    "code": 0,
    "msg": "",
    "data": null
}
```
