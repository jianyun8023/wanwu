# 草稿态对话历史API

获取指定智能体的**草稿态**对话历史消息。草稿态每个智能体仅维护一条会话，因此通过 `uuid` 直接定位即可，无需传 `conversation_id`。

> 草稿态对话调用接口为 [3.草稿态对话API.md](3.草稿态对话API.md)；本接口仅用于查询历史。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/conversation/draft/detail` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| uuid | 是 | string | 智能体UUID |
| pageNo | 是 | int | 页码，从 1 开始 |
| pageSize | 是 | int | 每页条数，从 1 开始 |

## 响应参数

响应结构与 [13.对话历史消息API.md](13.对话历史消息API.md) 一致。当草稿会话不存在时返回空列表。

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/conversation/draft/detail?uuid=2004010127610351679&pageNo=1&pageSize=20' \
--header 'Authorization: Bearer <API Key>'
```
