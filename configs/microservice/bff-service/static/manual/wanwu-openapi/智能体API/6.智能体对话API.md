# 智能体对话API

向智能体发起问答。支持流式和非流式两种模式。如需在对话中携带文件，请先调用**文件上传API**获取文件信息后再传入。

> **注意：** 本接口基于已发布版本进行问答。如需在发布前进行调试，请使用**草稿态对话API**。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/chat` |
| 字符编码 | UTF-8 |
| 请求类型 | POST |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json` |
| 响应格式 | 非流式：`Content-Type: application/json` / 流式：`Content-Type: text/event-stream` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| uuid | 是 | string | 智能体UUID |
| conversation_id | 否 | string | 会话ID，由创建对话接口返回；不传时为单轮对话，无历史上下文 |
| query | 是 | string | 用户输入的问题或指令 |
| stream | 否 | bool | 是否流式返回，默认为 false（非流式） |
| file_info | 否 | array | 随对话携带的文件列表，文件信息需通过文件上传API提前获取。**当前一次仅支持 1 个文件**，array 形态为未来多文件能力预留；传入超过 1 个会返回参数错误 |

### file_info

文件信息来源于**文件上传API**响应结果中的 `data.files`，字段对应关系如下：

| Parameter | Required | Type | Instructions | 对应上传响应字段 |
| --------- | -------- | ---- | ---- | ---- |
| fileName | 是 | string | 上传后的文件名称 | `fileId` |
| fileSize | 是 | int | 文件大小（字节） | `fileSize` |
| fileUrl | 是 | string | 文件完整访问路径 | `filePath` |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| message | string | 提示信息 |
| response | string | 模型回答文本 |
| search_list | array | 知识库检索结果列表 |
| gen_file_url_list | array | 模型生成的文件URL列表 |
| history | array | 对话历史 |
| usage | object | Token 用量统计 |
| finish | int | 仅流式返回中存在。结束原因：0-未结束，1-正常结束，2-超长度截断，3-异常结束，4-命中安全护栏 |

### search_list

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| kb_name | string | 知识库名称 |
| snippet | string | 命中的知识片段内容 |
| title | string | 来源文件标题 |

### gen_file_url_list

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| output_file_url | string | 模型生成文件的访问URL |

### history

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| query | string | 用户输入 |
| response | string | 模型回答 |

### usage

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| prompt_tokens | int | 输入 Token 数 |
| completion_tokens | int | 输出 Token 数 |
| total_tokens | int | 总 Token 数 |

## 调用示例

### 流式（不带文件）

```shell
curl -k --no-buffer --location 'http://localhost:8081/service/api/openapi/v1/agent/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "uuid": "2004010127610351679",
    "conversation_id": "56",
    "query": "请一句话介绍DeepSeek",
    "stream": true
}'
```

### 流式（带文件）

> 使用前请先调用**文件上传API**获取文件信息，再将响应结果中的 `fileId`、`filePath`、`fileSize` 填入 `file_info` 对应字段。

```shell
curl -k --no-buffer --location 'http://localhost:8081/service/api/openapi/v1/agent/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "uuid": "2004010127610351679",
    "conversation_id": "56",
    "query": "请解析文档并输出全部内容，不要有任何修改和加工",
    "stream": true,
    "file_info": [
        {
            "fileName": "59bfaef5-8609-48ad-a69d-305fcd221e93.txt",
            "fileSize": 1,
            "fileUrl": "http://minio-wanwu:9000/file-upload/file-expire/59bfaef5-8609-48ad-a69d-305fcd221e93.txt"
        }
    ]
}'
```

### 非流式

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "uuid": "2004010127610351679",
    "conversation_id": "56",
    "query": "请一句话介绍DeepSeek",
    "stream": false
}'
```
