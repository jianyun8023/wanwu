# 对话历史消息API

获取指定**已发布对话**的历史消息列表（问答明细），分页返回。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/conversation/detail` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| conversation_id | 是 | string | 对话ID，由[创建对话API](5.创建对话API.md)或[对话列表API](12.对话列表API.md)获取 |
| pageNo | 是 | int | 页码，从 1 开始 |
| pageSize | 是 | int | 每页条数，从 1 开始 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| list | array | 历史消息列表 |
| total | int | 总数 |
| pageNo | int | 当前页码 |
| pageSize | int | 每页条数 |

### data.list[]

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| id | string | 消息ID，可作为删除单条消息的 `detail_id` |
| conversationId | string | 对话ID |
| assistantId | string | 智能体内部ID |
| prompt | string | 用户输入 |
| sysPrompt | string | 系统提示词 |
| response | string | 模型回答 |
| responseList | array | 多段回答列表（如多轮思考） |
| searchList | object | 知识库召回结果 |
| qa_type | int | 问答类型 |
| createdBy | string | 创建人 |
| createdAt | int | 创建时间（Unix 毫秒） |
| updatedAt | int | 更新时间（Unix 毫秒） |
| requestFiles | array | 用户携带的文件列表 |
| responseFiles | array | 智能体生成的文件列表 |
| subConversationList | array | 子智能体/工具调用过程明细 |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/conversation/detail?conversation_id=56&pageNo=1&pageSize=20' \
--header 'Authorization: Bearer <API Key>'
```
