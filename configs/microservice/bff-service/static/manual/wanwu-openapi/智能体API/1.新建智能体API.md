# 新建智能体API

创建一个新的智能体应用，返回智能体的 UUID，后续所有接口均通过该 UUID 操作此智能体。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent` |
| 字符编码 | UTF-8 |
| 请求类型 | POST |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| name | 是 | string | 智能体名称 |
| category | 否 | int | 智能体类型：1-单智能体，2-多智能体，默认为1 |
| desc | 否 | string | 智能体描述 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| uuid | string | 新建智能体的UUID，用于后续所有接口传参 |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "name": "我的智能助手",
    "category": 1,
    "desc": "一个专业的AI助手"
}'
```

## 响应示例

```json
{
    "code": 0,
    "msg": "",
    "data": {
        "uuid": "2004010127610351679"
    }
}
```
