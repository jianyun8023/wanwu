# 智能体详情API

查询指定智能体的完整配置信息。通过 `published` 参数可选择查看**草稿态**（当前编辑中的配置）或**已发布态**（最新发布版本的配置）。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent/info` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| uuid | 是 | string | 智能体UUID |
| published | 否 | bool | `true`=已发布态，`false`=草稿态（默认） |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 智能体详情，见下方字段说明 |

### data 字段说明

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| uuid | string | 智能体UUID |
| name | string | 智能体名称 |
| desc | string | 智能体描述 |
| category | int | 智能体分类：1=单智能体，2=多智能体 |
| prologue | string | 开场白 |
| instructions | string | 系统提示词 |
| recommendQuestion | array[string] | 推荐问题列表 |
| modelConfig | object | 大模型配置，字段结构同[配置更新API](2.配置更新API.md)；`modelId` 返回模型 UUID |
| rerankConfig | object | Rerank模型配置，字段结构同 modelConfig；`modelId` 返回模型 UUID |
| knowledgeBaseConfig | object | 知识库配置，见下方结构说明 |
| safetyConfig | object | 敏感词配置 |
| visionConfig | object | 视觉配置 |
| memoryConfig | object | 记忆配置 |
| recommendConfig | object | 追问配置；`modelConfig.modelId` 返回模型 UUID |
| publishType | string | 发布范围：`public`/`organization`/`private`（草稿态为空） |
| createdAt | string | 创建时间 |
| updatedAt | string | 更新时间 |

### knowledgeBaseConfig 结构

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| knowledgebases | array | 已关联的知识库列表，每项包含 `id`、`name` 等字段 |
| config.maxHistory | int | 最长上下文轮数 |
| config.threshold | float | 过滤阈值 |
| config.topK | int | 召回知识条数 |
| config.matchType | string | 检索方式：`vector`（向量检索）、`text`（文本检索）、`mix`（混合检索） |
| config.priorityMatch | int | 混合检索权重模式：1-自定义权重，0-自动 |
| config.semanticsPriority | float | 混合检索时语义（向量）权重 |
| config.keywordPriority | float | 混合检索时关键词权重 |
| config.termWeight | float | 关键词系数 |
| config.termWeightEnable | bool | 是否启用关键词系数 |
| config.useGraph | bool | 是否启用知识图谱检索 |

## 调用示例

### 查看草稿态（默认）

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/info?uuid=2004010127610351679' \
--header 'Authorization: Bearer <API Key>'
```

### 查看已发布态

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/agent/info?uuid=2004010127610351679&published=true' \
--header 'Authorization: Bearer <API Key>'
```
