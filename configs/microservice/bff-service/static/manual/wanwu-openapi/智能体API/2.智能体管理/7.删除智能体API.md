# 删除智能体API

删除指定智能体，同时删除其已发布版本。**删除后不可恢复，请谨慎操作。**

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent` |
| 字符编码 | UTF-8 |
| 请求类型 | DELETE |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | 参数走 query string，无请求体 |
| 响应格式 | `Content-Type: application/json` |

## 请求参数（query）

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| uuid | 是 | string | 智能体UUID |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 空对象 |

## 调用示例

```shell
curl -k --location --request DELETE 'http://localhost:8081/service/api/openapi/v1/agent?uuid=2004010127610351679' \
--header 'Authorization: Bearer <API Key>'
```

## 响应示例

```json
{
    "code": 0,
    "msg": "",
    "data": null
}
```
