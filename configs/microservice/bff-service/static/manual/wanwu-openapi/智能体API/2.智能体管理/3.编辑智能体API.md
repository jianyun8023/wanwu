# 编辑智能体API

修改智能体的**基本信息**（名称、描述、图标）。通过 `assistantUuid` 定位智能体。

> 仅用于编辑基本信息；开场白、大模型、知识库等高级配置请使用 [配置更新API](2.配置更新API.md)。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/agent` |
| 字符编码 | UTF-8 |
| 请求类型 | PUT |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| assistantUuid | 是 | string | 智能体UUID |
| name | 是 | string | 智能体名称 |
| desc | 否 | string | 智能体描述 |
| avatar | 否 | object | 图标，详见 avatar |

### avatar

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| key | string | 图标存储 key，使用默认图标时传空字符串 |
| path | string | 图标访问路径，例如 `/v1/static/icon/agent-default-icon.png` |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 固定为 null |

## 调用示例

```shell
curl -k --location --request PUT 'http://localhost:8081/service/api/openapi/v1/agent' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "assistantUuid": "2004010127610351679",
    "name": "我的智能助手",
    "desc": "更新后的描述",
    "avatar": {
        "key": "",
        "path": "/v1/static/icon/agent-default-icon.png"
    }
}'
```

## 响应示例

```json
{
    "code": 0,
    "data": null,
    "msg": ""
}
```
