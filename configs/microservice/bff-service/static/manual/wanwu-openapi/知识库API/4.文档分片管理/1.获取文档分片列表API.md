# 获取文档分片列表API

## 知识库获取文档分片列表API
知识库获取文档分片列表支持查看导入文档的各分片内容相关的信息。

### 请求接口

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/list` |
| 调用方式 | 同步调用                                                      |
| 字符编码 | UTF-8                                                        |
| 请求类型 | POST                                                         |
| 鉴权方式 | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json`                             |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

| Parameter    | Required  | Type   | Instructions                                                 |
|---------------|----| ------ |--------------------------------------------------------------|
| docId | 是  | string | 文档Id，通过知识库获取的文档列表中每个文档的Id，通过此Id查询文档分片信息                      |
| pageSize    | 否  | int | 分页大小                                                         |
| pageNo      | 否  | int | 分页序号                                                         |

### 响应参数

| Parameter          | Type         | Instructions   |
|--------------------| ------------ |--------------|
| code               | int          | 状态码        |
| data               | object array | 返回数据   |
| msg                | string       | 返回信息       |

#### data中详细字段

| Parameter             | Type         | Instructions   |
|--------------------   |--------------| ---------- |
| fileName              | string       | 文件名称   |
| pageTotal             | int          | 总页数   |
| segmentTotalNum       | int          | 分段数量   |
| maxSegmentSize        | int          | 设置最大长度   |
| segmentType           | string       | 分段方式 0:自动分段 1:自定义分段   |
| uploadTime            | string       | 上传时间   |
| splitter              | string       | 分隔符（只有自定义分段必填） |
| metaDataList          | object array | 文档元数据列表   |
| contentList           | object array | 分段内容列表   |
| segmentImportStatus   | string       | 分段导入状态描述   |
| segmentMethod         | string       | 分段方式 0:通用分段，1:父子分段   |
| docAnalyzerText       | object array | 文档解析类型列表   |
| downloadUrl           | string       | 文档对外下载地址   |

#### metaDataList中详细字段

| Parameter             | Type         | Instructions   |
|--------------------   |--------------| ---------- |
| metaId                | string       | 元数据id   |
| metaKey               | string       | 元数据key   |
| metaValue             | string       | 元数据值   |
| metaValueType         | string       | 元数据值类型 number/time/string   |
| metaRule              | string       | 正则表达式   |

#### contentList中详细字段

| Parameter             | Type         | Instructions   |
|--------------------   |--------------| ---------- |
| content               | string       | 分段内容   |
| available             | bool         | 是否可用   |
| contentId             | string       | 分段内容id   |
| contentNum            | int          | 分段序号   |
| labels                | string array | 标签列表   |
| isParent              | bool         | 是否为父分段 true:父分段 false:通用分段或子分段   |
| childNum              | int          | 子分段数量   |

#### docAnalyzerText中详细字段

| Parameter             | Type         | Instructions   |
|--------------------   |--------------| ---------- |
| text                  | string       | 解析类型文本 如：文字提取/OCR解析/模型解析   |
| displayName           | string       | 显示名称   |
| tags                  | object array | 标签列表   |

### 请求示例

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/list?docId=2061726711199830016&pageNo=1&pageSize=8' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}' 
```
