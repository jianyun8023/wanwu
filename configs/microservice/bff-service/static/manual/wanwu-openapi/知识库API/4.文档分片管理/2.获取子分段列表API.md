# 获取子分段列表API

## 知识库获取子分段列表API
知识库获取子分段列表支持查看父子分段文档中指定父分段的子分段列表信息。

### 请求接口

| Type     | Instructions                                                 |
| -------- | ------------------------------------------------------------ |
| 方法     | Http                                                         |
| 请求URL  | 按照应用实际API根地址，例如：<br />`http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/child/list` |
| 调用方式 | 同步调用                                                      |
| 字符编码 | UTF-8                                                        |
| 请求类型 | GET                                                          |
| 鉴权方式 | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json`                             |
| 响应格式 | `Content-Type: application/json`                             |

### 请求参数

| Parameter    | Required  | Type   | Instructions                                                 |
|---------------|----| ------ |--------------------------------------------------------------|
| docId | 是  | string | 文档Id                      |
| contentId    | 是  | string | 父分段内容id                                                         |

### 响应参数

| Parameter          | Type         | Instructions   |
|--------------------| ------------ |--------------|
| code               | int          | 状态码        |
| data               | object       | 返回数据   |
| msg                | string       | 返回信息       |

#### data中详细字段

| Parameter             | Type         | Instructions   |
|--------------------   |--------------| ---------- |
| contentList           | object array | 子分段内容列表   |

#### contentList中详细字段

| Parameter             | Type         | Instructions   |
|--------------------   |--------------| ---------- |
| content               | string       | 子分段内容   |
| childId               | string       | 子分段id   |
| childNum              | int          | 子分段序号   |
| parentId              | string       | 父分段id   |

### 请求示例

```shell
curl --location 'http://localhost:8081/service/api/openapi/v1/knowledge/doc/segment/child/list?docId=2061726711199830016&contentId=68d24b24be7778c1e047b4c0e66c355c' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer {API Key}'
```

### 响应示例

```json
{
  "code": 0,
  "data": {
    "contentList": [
      {
        "content": "这是一段测试分段的内容1",
        "childId": "abc123def456",
        "childNum": 1,
        "parentId": "68d24b24be7778c1e047b4c0e66c355c"
      },
      {
        "content": "这是一段测试分段的内容2",
        "childId": "abc123def457",
        "childNum": 2,
        "parentId": "68d24b24be7778c1e047b4c0e66c355c"
      }
    ]
  },
  "msg": ""
}
```
