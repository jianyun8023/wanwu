# 对话流API

向 WGA 发送消息，通过 SSE（Server-Sent Events）流式返回 AG-UI 协议事件。这是 WGA 的核心交互接口。

> **注意：** 本接口仅返回 SSE 流式响应，不支持非流式。如需保留多轮上下文，请先通过**创建对话API**获取 threadId，后续对话传入同一 threadId。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/wga/conversation/chat` |
| 字符编码 | UTF-8 |
| 请求类型 | POST |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 请求格式 | `Content-Type: application/json` |
| 响应格式 | `Content-Type: text/event-stream` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| threadId | 是 | string | 对话 ID，由创建对话接口返回 |
| modelUuid | 是 | string | 模型 UUID，指定对话使用的大模型 |
| messages | 是 | array | 消息列表，至少包含一条 user 消息 |
| agentId | 否 | string | 子智能体 ID。不传或留空走 Supervisor 默认路由；传入指定子智能体（如 `PPT Agent`）则直连该智能体 |

### messages[]

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| id | 否 | string | 消息 ID，由客户端生成用于追踪 |
| role | 是 | string | 角色，当前仅支持 `user` |
| content | 是 | string 或 array | 消息内容 |

### content 格式

**纯文本：**

```json
"content": "帮我搜索AI最新进展"
```

**多模态（文本 + 文件）：**

```json
"content": [
    { "type": "text", "text": "读取该文件内容" },
    { "type": "binary", "mimeType": "application/octet-stream",
      "url": "http://minio-wanwu:9000/file-upload/.../file.md",
      "fileName": "README_CN.md" }
]
```

> 多模态元素说明：
> - `text`：文本内容，字段 `text`
> - `binary`：文件引用，字段 `url`（文件完整路径）、`fileName`（文件名）、`mimeType`（MIME 类型）
> - 文件 URL 需先通过 `/v1/file/upload/direct` 上传获取

## 响应格式（SSE 事件流）

每行格式：

```
data: {"type":"EVENT_TYPE","timestamp":1718443800000,...}
```

### 主要事件类型

| 事件类型 | 说明 | 关键字段 |
|---------|------|---------|
| `RUN_STARTED` | 运行开始 | `threadId`, `runId` |
| `RUN_FINISHED` | 运行正常结束 | `threadId`, `runId` |
| `RUN_ERROR` | 运行出错 | `message` |
| `TEXT_MESSAGE_START` | 文本消息开始 | `messageId`, `role` |
| `TEXT_MESSAGE_CONTENT` | 文本增量 | `messageId`, `delta` |
| `TEXT_MESSAGE_END` | 文本消息结束 | `messageId` |
| `REASONING_MESSAGE_START` | 推理开始 | `messageId` |
| `REASONING_MESSAGE_CONTENT` | 推理增量 | `messageId`, `delta` |
| `REASONING_MESSAGE_END` | 推理结束 | `messageId` |
| `TOOL_CALL_START` | 工具调用开始 | `toolCallId`, `toolCallName` |
| `TOOL_CALL_ARGS` | 工具参数（流式 JSON） | `toolCallId`, `delta` |
| `TOOL_CALL_END` | 工具调用结束 | `toolCallId` |
| `TOOL_CALL_RESULT` | 工具执行结果 | `toolCallId`, `content` |
| `ACTIVITY_SNAPSHOT` | 活动状态快照 | `activityType`, `content` |

### ACTIVITY_SNAPSHOT 详解

| activityType | 说明 | content 关键字段 |
|-------------|------|-----------------|
| `sub_agent` | 子智能体执行 | `agentName`, `status`（"started"/"finished"） |
| `workspace` | 工作区文件更新 | `fileCount`, `totalSize` |
| `question` | 智能体提问 | `questionId`, `status`, `questions` |

## 调用示例

### 纯文本对话（Supervisor 默认路由）

```shell
curl -k --no-buffer --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "threadId": "dbb35657-1ca7-4da9-9056-5b51fda085d1",
    "modelUuid": "2030935982164938752",
    "messages": [
        {
            "id": "msg-1",
            "role": "user",
            "content": "帮我搜索最近的AI技术进展并生成报告"
        }
    ]
}'
```

### 直连子智能体（PPT Agent）

```shell
curl -k --no-buffer --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "threadId": "dbb35657-1ca7-4da9-9056-5b51fda085d1",
    "modelUuid": "2030935982164938752",
    "agentId": "PPT Agent",
    "messages": [
        {
            "id": "msg-1",
            "role": "user",
            "content": "帮我生成一份2026年Q1季度销售汇报PPT"
        }
    ]
}'
```

### 带文件对话

```shell
curl -k --no-buffer --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/chat' \
--header 'Content-Type: application/json' \
--header 'Authorization: Bearer <API Key>' \
--data '{
    "threadId": "dbb35657-1ca7-4da9-9056-5b51fda085d1",
    "modelUuid": "2030935982164938752",
    "messages": [
        {
            "id": "msg-1",
            "role": "user",
            "content": [
                { "type": "text", "text": "读取文档内容" },
                {
                    "type": "binary",
                    "mimeType": "application/octet-stream",
                    "url": "http://minio-wanwu:9000/file-upload/file-expire/b13f6b38-bdc3-4021-afe0-cdb7555fd80c.md",
                    "fileName": "README_CN.md"
                }
            ]
        }
    ]
}'
```

## SSE 事件流示例

```
data: {"type":"RUN_STARTED","threadId":"dbb35657-1ca7-4da9-9056-5b51fda085d1","runId":"run-001"}

data: {"type":"REASONING_MESSAGE_START","messageId":"msg-reasoning-001"}

data: {"type":"REASONING_MESSAGE_CONTENT","messageId":"msg-reasoning-001","delta":"我需要先搜索AI技术进展..."}

data: {"type":"REASONING_MESSAGE_END","messageId":"msg-reasoning-001"}

data: {"type":"TOOL_CALL_START","toolCallId":"tc-001","toolCallName":"web_search"}

data: {"type":"TOOL_CALL_ARGS","toolCallId":"tc-001","delta":"{\"query\":\"AI技术进展 2026\"}"}

data: {"type":"TOOL_CALL_END","toolCallId":"tc-001"}

data: {"type":"TOOL_CALL_RESULT","toolCallId":"tc-001","content":"搜索结果..."}

data: {"type":"ACTIVITY_SNAPSHOT","activityType":"sub_agent","content":{"agentName":"Deep Research Agent","status":"started"}}

data: {"type":"ACTIVITY_SNAPSHOT","activityType":"sub_agent","content":{"agentName":"Deep Research Agent","status":"finished"}}

data: {"type":"TEXT_MESSAGE_START","messageId":"msg-002","role":"assistant"}

data: {"type":"TEXT_MESSAGE_CONTENT","messageId":"msg-002","delta":"根据搜索结果，以下是最近AI技术的进展..."}

data: {"type":"TEXT_MESSAGE_END","messageId":"msg-002"}

data: {"type":"RUN_FINISHED","threadId":"dbb35657-1ca7-4da9-9056-5b51fda085d1","runId":"run-001"}
```
