# 对话历史API

获取指定对话的 AG-UI 事件流历史，按运行（run）分组。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/wga/conversation/detail` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| threadId | 是 | string | 对话 ID |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| list | array | 运行记录列表，按时间倒序 |
| total | int | 总运行次数 |

### list[]

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| threadId | string | 对话 ID |
| runId | string | 运行 ID |
| createdAt | int64 | 运行创建时间（Unix 毫秒时间戳） |
| events | array\<object\> | 该次运行产生的 AG-UI 事件列表 |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/detail?threadId=dbb35657-1ca7-4da9-9056-5b51fda085d1' \
--header 'Authorization: Bearer <API Key>'
```

```json
    "code": 0,
    "msg": "",
    "data": {
        "list": [
            {
                "threadId": "dbb35657-1ca7-4da9-9056-5b51fda085d1",
                "runId": "run-001",
                "createdAt": 1718443800000,
                "events": [
                    { "type": "RUN_STARTED", "threadId": "dbb35657-1ca7-4da9-9056-5b51fda085d1", "runId": "run-001" },
                    { "type": "TEXT_MESSAGE_START", "messageId": "msg-1", "role": "assistant" },
                    { "type": "TEXT_MESSAGE_CONTENT", "messageId": "msg-1", "delta": "你好" },

    }
}
```
