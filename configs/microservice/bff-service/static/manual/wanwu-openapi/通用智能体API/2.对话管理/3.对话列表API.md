# 对话列表API

分页获取当前用户的所有 WGA 对话列表。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/wga/conversation/list` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| pageNo | 是 | int | 页码，从 1 开始 |
| pageSize | 是 | int | 每页条数 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| list | array | 对话列表 |
| total | int | 总条数 |

### list[]

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| threadId | string | 对话 ID |
| title | string | 对话标题 |
| createdAt | string | 创建时间（ISO 8601 格式） |
| isSkillConversation | bool | 是否为技能对话 |
| skillId | string | 技能 ID（仅技能对话） |
| previewId | string | 预览 ID（仅技能预览对话） |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/list?pageNo=1&pageSize=10' \
--header 'Authorization: Bearer <API Key>'
```

## 响应示例

```json
{
    "code": 0,
    "msg": "",
    "data": {
        "list": [
            {
                "threadId": "dbb35657-1ca7-4da9-9056-5b51fda085d1",
                "title": "数据分析对话",
                "createdAt": "2026-06-15T10:30:00Z",
                "isSkillConversation": false,
                "skillId": "",
                "previewId": ""
            }
        ],
        "total": 42
    }
}
```
