# 下载工作区文件API

下载工作区中指定的文件，返回二进制文件流。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/wga/conversation/workspace/download` |
| 字符编码 | UTF-8（URL 参数） |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/octet-stream` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| threadId | 是 | string | 对话 ID |
| runId | 是 | string | 运行 ID，从 SSE 流 `RUN_STARTED` 事件中获取 |
| path | 否 | string | 工作区内的文件路径，不填则下载整个工作区（ZIP 压缩包） |

## 响应格式

- 成功：`HTTP 200`，返回二进制文件流
- 响应头：`Content-Disposition: attachment; filename="<文件名>"`

## 调用示例

### 下载整个工作区

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/workspace/download?threadId=dbb35657-1ca7-4da9-9056-5b51fda085d1&runId=run-001' \
--header 'Authorization: Bearer <API Key>' \
--output workspace.zip
```

### 下载指定文件

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/workspace/download?threadId=dbb35657-1ca7-4da9-9056-5b51fda085d1&runId=run-001&path=output/quarterly_report.pptx' \
--header 'Authorization: Bearer <API Key>' \
--output quarterly_report.pptx
```
