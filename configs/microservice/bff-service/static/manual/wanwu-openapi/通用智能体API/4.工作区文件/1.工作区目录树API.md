# 工作区目录树API

获取指定对话的工作区文件目录结构。

## 请求接口

| Type | Instructions |
| ---- | ---- |
| 方法 | Http |
| 请求URL | 按照应用实际API根地址，例如：`http://localhost:8081/service/api/openapi/v1/wga/conversation/workspace` |
| 字符编码 | UTF-8 |
| 请求类型 | GET |
| 鉴权方式(header参数) | `Authorization: Bearer {API Key}` |
| 响应格式 | `Content-Type: application/json` |

## 请求参数

| Parameter | Required | Type | Instructions |
| --------- | -------- | ---- | ---- |
| threadId | 是 | string | 对话 ID |
| runId | 是 | string | 运行 ID，从 SSE 流 `RUN_STARTED` 事件中获取 |

## 响应参数

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| code | int | 状态码，0表示成功 |
| msg | string | 提示信息 |
| data | object | 响应数据 |

### data

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| threadId | string | 对话 ID |
| runId | string | 运行 ID |
| fileCount | int32 | 文件总数 |
| totalSize | int64 | 文件总大小（字节） |
| isDisplay | bool | 是否有内容可展示 |
| path | string | 当前路径 |
| files | array | 文件/目录树 |

### files[]（递归结构）

| Parameter | Type | Instructions |
| --------- | ---- | ---- |
| name | string | 文件/目录名称 |
| type | string | 类型：`file` / `directory` |
| size | int64 | 文件大小，仅 type=file 时有值 |
| mimeType | string | MIME 类型，仅 type=file 时有值 |
| children | array | 子文件/目录列表，仅 type=directory 时有值 |

## 调用示例

```shell
curl -k --location 'http://localhost:8081/service/api/openapi/v1/wga/conversation/workspace?threadId=dbb35657-1ca7-4da9-9056-5b51fda085d1&runId=run-001' \
--header 'Authorization: Bearer <API Key>'
```
