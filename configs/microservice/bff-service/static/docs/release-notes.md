# ✨v0.5.8 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。ZZ


## 🚀 新功能

- **本体智能体：**
  - 数据连接支持Dameng数据库类型。
  - 数字员工上线【供应链数据采集专员】支持从图片或文档中的供应链业务数据自动提取导入数据库

- **智能体：**
  - 支持链接保持


## 📈增强功能

- **用户体验：**
  - 智能体添加工具可查看描述
  - 智能体命中知识库，可展示卡片


## 🛠️ 错误修复

- 通用智能体对话列表无法滚动下拉
- 知识库同时添加多个元数据，只保留了第一个
