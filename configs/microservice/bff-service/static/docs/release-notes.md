# ✨v0.5.9 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。ZZ


## 🚀 新功能

- **本体智能体：**
  - 数据连接的数据视图模块上线【自定义视图】功能，支持将多张表的字段数据关联/合并至新视图，在知识网络对象类配置中可引用【自定义视图】实现将N张表字段映射至同一对象属性上的复杂场景。
  - 知识网络的任务管理模块上线【定时构建管理】功能，通过自定义定时任务将数据源更新后的数据定时同步至本体知识网络中，实现数据一致性更新。

- **智能体：**
  - 支持发布后隐藏知识库出处

- **模型管理：**
  - 模型体验支持链接保持

- **通用智能体：**
  - 支持链接保持
  - 支持OpenAPI调用

- **平台配置：**
  - 支持侧边导航栏收缩logo配置


## 📈增强功能

- **智能体：**
  - 调用Skill过程优化
  - 生成可下载文件优化
  - 追问提示词编辑优化

- **知识问答：**
  - 选择图文问答类模型时，上传图片大小与模型导入时配置的图片大小联动

- **用户体验：**
  - 通用智能体样式优化

- **Skill：**
  - 一键转化为Skill功能仅针对已发布应用生效


## 🛠️ 错误修复

- **OpenAPI修复：**
  - 创建知识库可以选择非Embedding模型
  - 创建建智能体：name无限制+category负数问题
  - 智能体配置缺少参数校验：长度/范围/枚举/数组元素
  - 智能体非流式对话添加知识库可下载链接
  - 智能体缺少追问
