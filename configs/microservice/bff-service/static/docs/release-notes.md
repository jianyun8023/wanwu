# ✨v0.5.6 有哪些新功能？✨

在此版本中，我们添加了大量新功能、提升了用户使用时的交互体验，并修复了用户反馈的一些问题，希望能为您带来更好的使用感受。

## 🚀 新功能

- **知识库：**
  - 支持搜索元数据类型为时间的元数据值
  - 知识图谱构建模型支持选择图文问答类模型
  - 问答库支持查看基础配置
  - 知识库支持文件原文预览
  - 知识库检索节点中返回的download_link已转为对外链接
- **智能体：**
  - 模型配置支持重置
  - 图文问答类模型支持工具调用
  - 历史记录支持后端滚动删除
- **通用智能体：**
  - 新增数字员工
- **Skill：**
  - 支持在线编辑与git管理
- **本体智能体：**
  - 新增数字员工
- **资源库：**
  - 新增MCP鉴权
- **工作流：**
  - 用API调用工作流，支持查看详细报错信息，包含报错节点信息
- **平台配置：**
  - 新增通用智能体名称、logo、欢迎语配置
- **模型体验：**
  - 图文问答模型支持上传图片问答

## 📈增强功能

- **用户体验：**
  - 侧边栏样式升级
  - 智能体模型配置，最大标记和maxtoken保持一致
  - 通用智能体界面布局优化

## 🛠️ 错误修复

- 智能体上传文件页面卡死
